S3ModelFetcher

class S3ModelFetcher(httpClient: HttpClient, dispatcherProvider: DispatcherProvider = StandardDispatchers, fileHashCalculator: Sha256FileHashCalculator = StandardSha256FileHashCalculator) : ModelFetcher

Fetches a deep filter model from an S3 bucket, verifies its integrity, and saves it locally. This class implements the ModelFetcher interface.

The model is downloaded from a predefined URL and its integrity is checked against a SHA-256 checksum downloaded from a corresponding checksum file URL.
